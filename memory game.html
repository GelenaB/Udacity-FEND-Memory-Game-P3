About the Game

Memory Game contains a deck of 16 cards shuffled in a random order, 8 of which are identical. Your task is to find and match these identical cards with the least amount of moves possible.

Score Rating

Three stars: up to 15 moves

Two stars: 16-24 moves

One star: over 25 moves
